Reading list app
================

https://fierce-depths-8280.herokuapp.com/

Simple Rails 4 app that lets a user keep track of books they are reading.

Currently features:
  * Add a new book (ajax)
  * Search and filter book list
  * Use Amazon ID to pull book image
